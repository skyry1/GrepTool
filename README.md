# GrepTool
自作ファイル検索アプリ兼VB.NET開発練習

## Dependency
 -  .NET Framework 4.7.2
 -  VisualStudio2019
 
## Usage
### アプリを使いたい方
Releaseフォルダをダウンロードしてご利用ください。
### 開発する方
binフォルダとsourceフォルダをダウンロードして下さい。
